Project created to learn how to program in GDScript and get used to Godot as a whole;

As the name suggests, the game is a Tactical RPG with no real depth, focusing on testing and showcasing standard mechanics of the genre rather than providing a fun experience, such as:
- Classes with different skills;
- Grid movement; 
- Different map objectives;
- Experience gain resulting in level ups and stat changes;

Currently has placeholder art and freely licensed audio;

(Last stable version on release; more recent changes might contain bugs)
